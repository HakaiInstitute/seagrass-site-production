# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [v1.2.0] - 2022-12-02
### Notes ###
- This is the third release of the Hakai Seagrass Site Level Production data package, to include new data from 2022. Data now spans the years 2015 to 2022. Previous years' data (i.e. 2014) are not currently included due to differences in spatial sampling design.
### Changed ###
- changelog.txt to track future changes to this project
- data_dictionary.csv for defining variables found within datasets
- Seagrass Site-Level Production Metadata record
- seagrass_density_survey.csv containing density survey data from 2015 - 2022
- seagrass_habitat_survey.csv containing density survey data from 2015 - 2022

## [v1.1.0] - 2022-01-12
### Notes ###
- This is the second release of the Hakai Seagrass Site Level Production data package, to include new data from 2020 and 2021. This update also includes code fixes for the data QC process. Data now spans the years 2015 to 2021. Previous years' data (i.e. 2014) are not currently included due to differences in spatial sampling design.
### Changed ###
- changelog.txt to track future changes to this project
- data_dictionary.csv for defining variables found within datasets
- Seagrass Site-Level Production Metadata record
- Hakai Seagrass Protocol for in-depth explanation of methods
- seagrass_density_survey.csv containing density survey data from 2015 - 2021
- seagrass_habitat_survey.csv containing density survey data from 2015 - 2021

## [v1.0.0] - 2019-06-27
### Notes ###
- This is the initial release of the Seagrass Site-Level Production dataset, currently containing data from 2015 to the early 2019
- Due to changes in survey design and data management protocols, this initial release will include data starting May 2015. A later update (date TBD) will include 2014 data that will be restructured to match current survey designs and data structures
### Added ###
- changelog.txt to track future changes to this project
- data_dictionary.csv for defining variables found within datasets
- Seagrass Site-Level Production Metadata record
- Hakai Seagrass Protocol for in-depth explanation of methods
- seagrass_density_survey.csv containing density survey data from 2015 - 2019
- seagrass_habitat_survey.csv containing density survey data from 2015 - 2019
